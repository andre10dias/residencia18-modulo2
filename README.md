# residencia18-modulo2

### .NET

> Criar projeto: 

```
dotnet new console
```

> Buildar: 

```
dotnet build
```

> Executar: 

```
dornet run
```

### Angular

> Criar projeto: 

```
ng new minhaApp --no-standalone --routing --ssr=false
```

> Adicionar o angular matirial: 

```
ng add @angular/material
```

> Buildar: 

```
ng serve
```

> Buildar utilizando as configurações do package.json: 

```
npm start
```
